Punjab Kings Store UI Issue Report

Date: 2026-03-11

Base URL: https://store.punjabkingsipl.in/

Focus: UI alignment, formatting, wording, and spelling/copy issues.

Reviewed pages included the home page, products listing, brands, order tracking, about us, contact us, one representative product detail page, and mobile checks for the home page and PDP. No major spelling errors were observed on the About Us, Contact Us, Privacy Policy, Shipping Policy, Return Policy, or Terms pages during this pass, but the issues below were visible and reproducible in the current UI.

Findings

Severity Type URL Issue Recommendation Evidence
High Copy / English /order-tracking The heading reads Where is Order Id?, which is awkward English and inconsistent capitalization. The same phrase is repeated again under the CTA, making the page feel unfinished. Change the primary copy to something clearer such as Where is your order ID? or Track your order, and remove the duplicate line below the button. ui-order-tracking-page.png
Medium Copy / Grammar /product/pbks-varsity-jacket-17632075 The pricing helper text says Price inclusive of all tax. That is grammatically incorrect. Replace it with Price inclusive of all taxes. ui-pdp-tax-copy.png
Medium Copy / Taxonomy leak /products The filter rail exposes unclear or internal-looking labels such as Size Depth, L1 Category, and L2 Category. These do not read like customer-facing filter names. Rename these filters to business-friendly labels or hide them if they are internal taxonomy fields. ui-products-filter-size-depth.png
ui-products-filter-l1-category.png
ui-products-filter-l2-category.png
Medium Responsive UI / Readability / (mobile) On the mobile home page, the footer links and legal text render very small and feel cramped. They are readable only with effort and do not look properly spaced for a production mobile footer. Increase footer font size and vertical spacing on mobile, and make link groups easier to scan and tap. store-punjabkingsipl-in-mobile.png
Low Layout / Alignment /brands The Brands page looks visually incomplete: a single oversized logo/card is pinned to the left while most of the page remains empty. The content does not feel centered or balanced relative to the available canvas. Rework the page grid so a single-brand state still feels intentional, or use a narrower content container and centered alignment. store-punjabkingsipl-in-brands.png
Low Layout / Visual balance /order-tracking The order-tracking module occupies a small area near the top of the page, leaving a large empty content gap before the footer. The page looks under-designed and visually unbalanced. Increase useful supporting content or tighten the page height so the form feels intentionally placed rather than floating in empty space. ui-order-tracking-page.png
